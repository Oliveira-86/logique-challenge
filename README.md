### Technologies

##### - ReactJS
##### - Spring-boot

## How it works

### Backend

##### Install dependencies
    
    $ ./mvnw clean install -DskipTests 

##### Run the project

   $ ./mvnw spring-boot:run
    
### Frontend

##### Install dependencies
    
    $ yarn install

##### Run the project

    $ expo start
